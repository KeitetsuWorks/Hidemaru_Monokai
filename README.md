Monokai for 秀丸エディタ
========


## Description

秀丸エディタをMonokai風の配色するための色設定ファイルと，C/C++のソースコードをシンタックスハイライト表示するためのキーワード設定ファイルです．

* [秀丸エディタ](http://hide.maruo.co.jp/software/hidemaru.html)
